Demostración de la nueva API REST integrada en el sistema POS.

Datos de la API

Haz clic en un botón para cargar datos...

Información de la API

URL Base

/api/

Métodos Soportados

GET POST PUT DELETE

Endpoints Principales

  • /api/mesas - Gestión de mesas
  • /api/productos - Gestión de productos
  • /api/ordenes - Gestión de órdenes
  • /api/categorias - Categorías
  • /api/estadisticas - Estadísticas

Ejemplos de Código JavaScript

Obtener Datos (GET)

// Obtener todas las mesas
fetch('/api/mesas')
  .then(response => response.json())
  .then(data => {
    if (data.success) {
      console.log(data.data);
    }
  });

Crear Datos (POST)

// Crear nueva mesa
fetch('/api/mesas', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    nombre: 'Mesa VIP'
  })
})
.then(response => response.json())
.then(data => console.log(data));
